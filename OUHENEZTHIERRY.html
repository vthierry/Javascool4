Modifications structurelles:
  1. [développement de jvs4] Les sources de javascool core & builder sont regroupés dans un projet netbean unique dans svn:javascool/work
    1. avec un build.xml unique qui génère tous les jars.
    2. thierry a pas encore intégré ce qui concerne le site web (voir ci dessous)
    3. Javas'Cool Core /Builder est réptutée ok et les mécanismes sont basés sur trois objets
      1. org.javascool.gui.Desktop qui définir les functions d'interaction avec l'interface graphique de JavaScool.
      2. org.javascool.core.ProgletEngine qui implémente les mécanismes de (compilation, exécution, proglet, ..) JavaScool.
      3. org.javascool.builder.ProgletsBuilder qui intègre tous les mécanismes de compilation de proglets
  2. [organisation du code] Le code java est complètement documenté en français
    1. (notre façon de coder doit aussi servir d'exemple puisque nos sources sont visibles)

    2. les fichiers package.html / package.txt sont des mémos de description.
    3. Note: le code a été annoté (pour l'instant sous forme de commentaires) dans la perspective de le préprocessuer avec atp
