Procédures

L'option de calcul "procedure" de l'onglet Principal permet de créer (dans le langage sélectionné) une procédure déduite des variables indiquées dans les champs "entrées" et "sorties".


Après avoir complété l'onglet Principal (en remplissant ou en ne remplissant pas le champ "sorties" suivant le langage choisi), cliquez sur le bouton Nouveau et vérifiez que le logiciel crée :

  • une procédure avec quatre paramètres (quantite, prixUnitaire, prixTotal, remise) dans le premier cas.
  • une procédure avec deux paramètres (quantite, prixUnitaire) dans le second cas.

A titre indicatif, voici comment vous pouvez compléter la procédure :

  • dans le premier cas, le prix total et la remise sont affichés dans le programme principal (exemple Visual Basic).
  • dans le second cas, le prix total et la remise sont affichés dans la procédure (exemple Javascool).
