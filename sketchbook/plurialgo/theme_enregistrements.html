Enregistrements

Avec l'onglet Principal et son bouton Nouveau, il est facile de définir de nouveaux types en effectuant des regroupements en enregistrements (ou en classes), comme par exemple :

  • un type Article regroupant le nom et le prix unitaire (télécharger).
  • un type Point avec deux coordonnées (télécharger).

Les composants du type Article sont précisés dans le champ "regroupement", après le symbole ":" suivant le nom du type. Dans le cas du type Point, les composants (x, y) sont déduits : à partir des variables numérotées (x1, y1, x2, y2), deux variables (objet1 et objet2) de type Point sont créées.

Si vous disposez déjà d'un programme (sans enregistrement), vous pouvez le reformuler en introduisant des enregistrements :

  1. Téléchargez le programme initial (exemple 1, exemple 2).
  2. Complétez l'onglet "Principal" (exemple 1, exemple 2), en fixant également le langage du programme final (pouvant différer de celui du programme initial).
  3. Cliquez sur le bouton Reformuler.
