Enregistrements

Avec l'onglet Principal et son bouton Nouveau, il est facile de définir de nouveaux types en effectuant des regroupements en enregistrements (ou en classes), comme par exemple :

  • un type Article regroupant le nom et le prix unitaire (télécharger).
  • un type Point avec deux coordonnées (télécharger).

Les composants de type Article sont précisés dans le champ "regroupement", après le symbole ":" suivant le nom du type. Dans le cas du type Point, les composants (x, y) sont déduits : à partir des variables numérotées (x1, y1, x2, y2), deux variables (objet1 et objet2) de type Point sont créées.

Si vous disposez déjà d'un programme Javascool sans sous-programme, vous pouvez le reformuler en introduisant des enregistrements (en Javascool ou dans un autre langage) :

  1. Téléchargez si nécessaire le programme Javascool (exemple 1, exemple 2).
  2. Complétez l'onglet "Principal" (exemple 1, exemple 2), puis cliquez sur le bouton Reformuler.

Le mécanisme de reformulation fonctionne également si l'on part d'un programme Algobox, Larp, Python ou Visual Basic.
