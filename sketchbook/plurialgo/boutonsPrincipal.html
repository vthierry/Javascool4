Boutons de l'onglet Principal

  • le bouton Nouveau permet de créer, dans un nouvel onglet de l'éditeur, un squelette de programme (voir variables ou fonctions).
  • le bouton Insérer permet d'insérer, dans le programme courant de l'éditeur, une déclaration de variables (exemple), une saisie (exemple), un affichage (exemple) ou un sous-programme (exemple).
  • le bouton Effacer efface le contenu de l'onglet Principal.
  • le bouton Traduire permet d'obtenir la traduction, dans un autre langage, du programme courant de l'éditeur.
  • le bouton Reformuler permet de reformuler le programme courant de l'éditeur en un programme avec fonctions, objets...
