Traduction de programmes Xcas

Lors de la traduction de programmes Xcas, il faut préciser le type des variables :

  1. Téléchargez le programme Xcas à traduire (exemple 1, exemple 2, exemple 3).
  2. Précisez le type des variables (exemple 1, exemple 2, exemple 3) dans l'onglet Principal.
  3. Fixez le langage de traduction (Javascool, Javascript...) et cliquez sur le bouton Traduire dans l'onglet Principal.

Le traducteur accepte la syntaxe en français (exemple) ou en anglais (exemple), à condition de respecter certaines contraintes illustrées dans ces deux exemples.

Création de programmes Xcas

PluriAlgo peut aussi vous aider à créer des programmes Xcas (exemple) :

  1. Fixez Xcas comme langage de développement dans l'onglet "Principal".
  2. Complétez l'onglet "Principal" (exemple), puis cliquez sur le bouton Nouveau.
  3. Complétez l'onglet "Si" (exemple), puis cliquez sur le bouton Insérer (après avoir fixé la position d'insertion dans l'éditeur).
  4. Complétez l'onglet "Boucles" (exemple), puis cliquez sur le bouton Transformer 1-n.

Il est possible de prolonger cet exemple en introduisant une fonction calculant le taux de remise à partir de la quantité :

  1. Sélectionnez la portion de code calculant le taux de remise (exemple).
  2. Complétez l'onglet "Principal" (exemple), puis cliquez sur le bouton Reformuler.
