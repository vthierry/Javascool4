Boutons de l'onglet Boucles

  • le bouton Créer permet de créer, dans un nouvel onglet de l'éditeur, un programme avec une boucle (exemple 1, exemple 2).
  • le bouton Insérer permet d'insérer, dans le programme courant de l'éditeur, une boucle (exemple 1, exemple 2).
  • le bouton Effacer réinitialise le contenu de l'onglet Boucle.
  • le bouton Transformer 1-n encapsule dans une boucle le programme courant (ou une partie) de l'éditeur (exemples).
