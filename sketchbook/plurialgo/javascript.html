Javascript

Le traducteur de PluriAlgo permet de passer automatiquement d'un programme Javascool (Larp, Python ou Algobox) à un programme Javascript.

Ce code Javascript doit ensuite être exécuté par un navigateur web, par exemple à partir du menu contextuel de l'éditeur Javascool (clic droit, puis commande Web) qui ouvre le navigateur par défaut de votre ordinateur.
