Traduction de programmes Algobox

De nombreux lycéens débutent la programmation avec Algobox. PluriAlgo s'y adapte en permettant la traduction de programmes Algobox :

  1. Téléchargez le programme Algobox à traduire, soit sous sa forme algorithmique (exemple 1, exemple 2), soit tel qu'il est stocké (exemple 1, exemple 2).
  2. Dans l'onglet "Principal", fixez le langage de traduction (Javascool, Javascript...) et cliquez sur le bouton Traduire.

La commande Ouvrir un fichier de Javascool peut être utilisée pour ouvrir un fichier Algobox afin de le traduire.

Création de programmes Algobox

PluriAlgo peut aussi vous aider à créer des programmes Algobox (exemple) :

  1. Fixez Algobox comme langage de développement dans l'onglet "Principal".
  2. Complétez l'onglet "Principal" (exemple), puis cliquez sur le bouton Nouveau.
  3. Complétez l'onglet "Si" (exemple), puis cliquez sur le bouton Insérer (après avoir fixé la position d'insertion dans l'éditeur).
  4. Complétez l'onglet "Boucles" (exemple), puis cliquez sur le bouton Transformer.

Le menu contextuel de l'éditeur Javascool (clic droit) permet d'exporter le code obtenu dans l'environnement Algobox (commande Algobox) ... s'il est installé !
