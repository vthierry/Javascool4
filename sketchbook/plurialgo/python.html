Traduction de programmes Python

Lors de la traduction de programmes Python, il faut préciser le type des variables :

  1. Téléchargez le programme Python à traduire (exemple 1, exemple 2, exemple 3).
  2. Précisez le type des variables (exemple 1, exemple 2, exemple 3) dans l'onglet Principal.
  3. Fixez le langage de traduction (Javascool, Javascript...) et cliquez sur le bouton Traduire dans l'onglet Principal.

Une limite du traducteur est sa difficulté à détecter la fin d'une instruction conditionnelle ou itérative lorsqu'il analyse le programme Python à traduire : c'est pourquoi les commentaires #end figurent ici dans les 3 exemples.

Une autre limite apparaît clairement quand on traduit l'exemple 2 en Javascool : les variables locales (prixTotal, remise) de la fonction restent à déclarer.

Création de programmes Python

PluriAlgo peut aussi vous aider à créer des programmes Python (exemple) :

  1. Fixez Python comme langage de développement dans l'onglet "Principal".
  2. Complétez l'onglet "Principal" (exemple), puis cliquez sur le bouton Nouveau.
  3. Complétez l'onglet "Si" (exemple), puis cliquez sur le bouton Insérer (après avoir fixé la position d'insertion dans l'éditeur).
  4. Complétez l'onglet "Boucles" (exemple), puis cliquez sur le bouton Transformer.

Il est possible de prolonger cet exemple en introduisant une fonction calculant le taux de remise à partir de la quantité :

  1. Sélectionnez la portion de code calculant le taux de remise (exemple).
  2. Complétez l'onglet "Principal" (exemple), puis cliquez sur le bouton Reformuler.

Le menu contextuel de l'éditeur Javascool (clic droit) permet d'exporter le code obtenu dans un environnement de développement Python, sous réserve que votre système d'exploitation l'associe à l'extension de fichier ".py", ce qui est le cas si vous utilisez AmiensPython.
