Boucles

Si l'onglet "Resultats" contient un programme Javascool (ou Visual Basic), le menu contextuel (clic droit) permet d'ajouter une boucle (Pour ou Tantque). Mais le véritable apport du logiciel sur ce sujet est le mécanisme de "transformation 1-n", le "1" signifiant que le programme initial sera répété plusieurs fois (d'où le "n") :

  1. Dans l'onglet "Résultats", copiez votre programme Javascool (exemple 1, exemple 2).
  2. Dans l'onglet "Résultats", cliquez sur le bouton Transformer 1-n.
