Variables : déclaration, typage et entrées-sorties

L'onglet Principal permet de créer automatiquement des squelettes de programmes, à partir d'informations telles que les variables d'entrée ou de sortie et de leur type (entier, réel, texte...) : par exemple (télécharger), vous pouvez calculer un prix total (variable de sortie) à partir d'un nombre d'articles et d'un prix unitaire (variables d'entrée). Il reste alors à cliquer sur le bouton Nouveau, après avoir fixé le langage de programmation : javascool, visual basic (vb), larp, javascript...

Je vous invite à choisir le langage Javascool : non seulement vous pourrez tester le programme (à compléter) dans l'environnement Javascool, mais vous pourrez aussi le traduire automatiquement dans d'autres langages de programmation usuels.

Remarque : pour pouvoir compiler et tester le programme javascool, il faut le transférer de l'onglet "Résultats" de la proglet vers l'éditeur de fichiers Javascool ; il suffit pour cela de faire apparaître le menu contextuel avec un clic droit et d'effectuer un copier-coller.
