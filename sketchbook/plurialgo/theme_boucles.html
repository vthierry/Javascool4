Boucles

L'onglet "Boucles" permet, comme son nom le suggère, d'introduire facilement des boucles (dans le langage fixé dans l'onglet Principal) .

Vous pouvez par exemple l'utiliser pour introduire une boucle parcourant tous les entiers impairs de 1 à 99 : complétez l'onglet comme indiqué ci-dessous (télécharger), puis cliquez sur le bouton Créer (ou Insérer). Grâce à l'option "sommation", le code obtenu calcule aussi la somme des inverses : 1/1 + 1/3 + 1/5 + ... + 1/99.


Vous pouvez également l'utiliser pour itérer les instructions du programme courant de l'éditeur Javascool (exemple) : complétez l'onglet comme indiqué ci-dessous (télécharger), puis vérifiez que le bouton Transformer 1-n permet ici de passer d'un programme lisant une note à un programme lisant (et sommant) plusieurs notes (voir compléments pour en savoir plus...).
