Boucles

L'onglet "Boucles" permet, comme son nom le suggère, d'introduire facilement des boucles (dans le langage fixé dans l'onglet Principal) .

Vous pouvez par exemple l'utiliser pour introduire une boucle parcourant tous les entiers impairs de 1 à 99 : complétez l'onglet comme indiqué ci-dessous (télécharger), puis cliquez sur le bouton Créer (ou Insérer). Grâce à l'option "sommation", le code obtenu calcule aussi la somme des inverses : 1/1 + 1/3 + 1/5 + ... + 1/99.


Vous pouvez aussi créer une boucle répétant des instructions sélectionnées dans l'éditeur :

  1. Téléchargez le programme initial (exemple Javascool, Larp, Python), où les instructions à répéter (ici la saisie d'une note) doivent être sélectionnées.
  2. Complétez l'onglet "Boucles" : sommation (exemple 1), comptage (exemple 2), minimum et maximum (exemple 3), tantque (exemple 4).
  3. Cliquez sur le bouton Transformer, en indiquant auparavant le langage du programme initial dans l'onglet Principal.
