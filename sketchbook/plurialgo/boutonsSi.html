Boutons de l'onglet Si

  • le bouton Créer permet de créer, dans un nouvel onglet de l'éditeur, un programme avec une instruction conditionnelle (exemple).
  • le bouton Insérer permet d'insérer, dans le programme courant de l'éditeur, une instruction conditionnelle (exemple).
  • le bouton Effacer réinitialise le contenu de l'onglet Si.
