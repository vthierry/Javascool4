Javascool

Environnement Javascool

Après avoir écrit un programme Javascool (exemple), vous devez en faire contrôler la syntaxe (bouton Compiler) : s'il n'y a aucune erreur, un bouton Executer apparaît...


Si vous modifiez ensuite ce programme, vous devrez le recompiler avant de cliquer sur le bouton Executer : sinon, c'est le programme initial qui s'exécutera !

Création de programmes Javascool

PluriAlgo peut vous aider à créer des programmes Javascool (exemple) :

  1. Fixez Javascool comme langage de développement dans l'onglet "Principal".
  2. Complétez l'onglet "Si" (exemple), puis cliquez sur le bouton Créer.
  3. Complétez l'onglet "Principal" (exemple), puis cliquez sur le bouton Reformuler.
  4. Complétez l'onglet "Boucles" (exemple), puis cliquez sur le bouton Transformer 1-n.

Il serait possible d'introduire une variable "nom" à l'étape 3, mais vous pouvez encore le faire après l'étape 4 :

  • pour déclarer la variable "nom", complétez l'onglet Principal (exemple), puis cliquez sur le bouton Insérer (après avoir fixé la position d'insertion dans l'éditeur).
  • pour effectuer la saisie du nom de l'article, complétez l'onglet Principal (exemple), puis cliquez sur le bouton Insérer.

Ce bouton, très pratique, permet également de définir des sous-programmes (exemple).


Remarque : il y a aussi un bouton Insérer dans les onglets "Boucles" et "Si" (voir boutons).
