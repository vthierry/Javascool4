Traduction de programmes Larp

Lors de la traduction de programmes Larp, il faut préciser le type des variables :

  1. Téléchargez le programme Larp à traduire (exemple 1, exemple 2).
  2. Précisez le type des variables (exemple 1, exemple 2) dans l'onglet Principal.
  3. Fixez le langage de traduction (Javascool, Javascript...) et cliquez sur le bouton Traduire dans l'onglet Principal.

Création de programmes Larp

PluriAlgo peut aussi vous aider à créer des programmes Larp (exemple) :

  1. Fixez Larp comme langage de développement dans l'onglet "Principal".
  2. Complétez l'onglet "Principal" (exemple), puis cliquez sur le bouton Nouveau.
  3. Complétez l'onglet "Si" (exemple), puis cliquez sur le bouton Insérer (après avoir fixé la position d'insertion dans l'éditeur).
  4. Complétez l'onglet "Boucles" (exemple), puis cliquez sur le bouton Transformer 1-n.
